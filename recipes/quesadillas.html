Quesadillas

Quesadilla con queso

Receta para preparar unas ricas quesadillas con queso

Siguiendo estos pasos tendras una cena de lujo para atascarse deliciosamente.

Ingredientes

  • Tortillas
  • Queso

Receta

  1. Poner las tortillas en el comal
  2. Una vez que la tortilla este calientita agregar queso
  3. Cuando el queso este derretido retirar para comer o dejar para que se dore la tortilla al gusto
